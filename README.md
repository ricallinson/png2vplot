# jpg2vplot

Convert a jpg image to a vplot file

## Developer Setup

    cd $GOPATH
    git clone git@github.com:ricallinson/jpg2vplot.git ./src/github.com/ricallinson/jpg2vplot

## Install

    cd ./src/github.com/ricallinson/jpg2vplot
    go install

## Use

    jpg2vplot ./fixtures/test.jpg ./test.vplot

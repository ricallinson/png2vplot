# png2vplot

Convert a png image to a vplot file

## Developer Setup

    cd $GOPATH
    git clone git@github.com:ricallinson/png2vplot.git ./src/github.com/ricallinson/png2vplot

## Install

    cd ./src/github.com/ricallinson/png2vplot
    go install

## Use

    png2vplot ./fixtures/test.png ./test.vplot
